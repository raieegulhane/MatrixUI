Flex

Flex can be used as an convenient way of creating responsive layouts to arrange items in rows and columns.
Flex property can be applied to any container using display: flex;
To set direction of flow of items:
- Horizontal -- flex-direction: row;
- Verical -- flex-direction: column;

Flex Row

To apply flex propert to any box use class flex
flex-row class used to impart horizontal flow to the items in the box.

Layouts

This is how the content items will be laid out on the main axis.
You can the following classes to apply various justify-content properties to the flex items.
- flex-start: flex-start
- center: flex_justify-center
- flex-end: flex_justify-end
- space-between: flex_justify-sb
- space-around: flex_justify-sa
- space-evenly: flex_justify-se

Alignment

Flex property align-items decides how the flex items are laid out on the cross axis.
Following classes can be used:
- flex-start: flex_align-top
- center: flex_align-middle
- flex-end: flex_align-bottom
