Badge

Used to highlight the status of any item for quick recognization. It can also display message.

Default

Simple badge that can be edited as per need.
In the following example badge is shown on button, but it can be used with any item.
Use class badge.

Badge With Content

Can be used to display count or small message.
By default badge is round/oval in shape. Apply class badge-sq for squared corners and class badge-cr for rounded ends.

Badge On Text

Badge is placed on the corner of text or heading.
Both the text and badge must be placed inside the same container that has property position: relative.

Status Badge

Status badges are used often with avatar to show the status of user. For example - when user is online, the status badge will be green.
The avatar image/text and badge should be in same container that has position: relative
Apply class badge_status.
Additionally for circular avatars use class:
- .badge_status-cr-sm: for small avatar (avatar-sm)
- .badge_status-cr-md: for medium avatar (avatar-sm)
- .badge_status-cr-lg: for large avatar (avatar-sm)
For square avatar use class badge_status-sq

Colorful Indicators

By default the badge is red. Add following classes to change colors.
badge-orange, badge-yellow, badge-green, badge-cyan, badge-blue, badge-voilet and badge-gray.
Alternately badge color can be changed through inline styling or changing color property in badge class.
