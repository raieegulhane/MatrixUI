Input

Used for accepting input from the user.

Default

Use class input to style the component.
Additional classes:
- input-sq: For square ended input box.
- input-cr: For round ended input box.

Textbox

Used to accept multi-lined inputs. Use class input-textarea.

-

With Button

Provied user with a call to action after the input is entered.
The button and input box should be put inside a container with class container_input-btn.

Password

For entering passwords discretely. The input entered is not visible.
Additionally a button for toggeling between hihhen password and visible password can be added.
