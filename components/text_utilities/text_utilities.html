Heading Sizes

To use custom heading sizes following classes can be used:
- heading-lg: Large haeding
- heading-md: Medium haeding
- heading-sm: Small haeding
- heading-xs: Extra-small haeding

Text Sizes

To use custom text sizes following classes can be used:
- txt-lg: Large text
- txt-md: Medium text
- txt-sm: Small text
Default text has font-size: 1rem (16px).

Decoration And Styles

Classes used:
- txt-bold: Bold text
- txt-ittalic: Italic text
- txt-underline: Underline text
- txt-overline: Overline text
- txt-highlight: Highlighted text
- txt-linethrough: Line-through text
- txt-primary: Primary color text
- txt-gray: Gray text
