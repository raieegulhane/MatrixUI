Avatar

Represents an user or a brand. It may display image, text, falback image etc.

Default

Use class avatar-square for square avatar and avatar-circle for round avatar. avatar-border class can be used to add border to default avatar. You can add alt to make sure avatar show pure text avatar when image loading fails.

Charachter Avatar

Avatar shows charachters instead of image.
Use class avatar-char. Color and font properties can be adjusted using this class.
