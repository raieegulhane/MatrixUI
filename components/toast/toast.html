Toast

Used for giving feedback to users after an action has taken place, or to request for some action. The toast will close itself when the close button is clicked, or after a timeout.

Default

Anything put inside toast_container will be displayed as toast.
Use class toast_default.

Status

Showcases the type of message in toast using different colors.
Classes used:
- toast-info: Informational toast
- toast-success: Success toast
- toast-warning: Warning toast
- toast-error: Error toast
