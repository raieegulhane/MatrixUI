Grids

Grids are an easy way of structuring webpages. Components of the website can be arranged into rows and columns to create complex layouts.

Basic Multi-Column Grids

Use class: grid.
Number of rows or columns in which a grid needs to be divided is given using grid-template-row or grid-template-column properties respectively.
Here examples of basic multi-column grids are shown. Classes used are:
- grid_2col
- grid_3col
- grid_4col
- grid_5col
for 2, 3, 4, and 5 columns, each taking one fraction (1fr) of the available space.

Holy Grail Layout

This layout is very commonly used on web. It containes a header at the top, a main content area with fixed-width navigation on the left, content in the middle and a fixed-width sidebar on the right and then a footer.
Use class grid_holygrail and then add your content in divs with class grid_header, grid_aside, grid_main, grid_footer.
