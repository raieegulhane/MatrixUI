List

Used to display items in a particular sequence. Has a wide range of applications.

Default

Has items listed in sequential manner. Any kind of content can be placed inside <li> tags.
Contents of <li> tag can styled using list-item class.

Unordered

Some commonly used styles for unordered list.
Following classes can be applied to <ul> tag:
- list_ul-ds: Disc shaped bulletes
- list_ul-sq: Square bulletes
- list_ul-cr: Circular bulletes

To apply styles to <ul> tag use class list-ul. To apply any background styles to list, put the list inside a container with class list_container and change the styles for this class.

Ordered

Some commonly used styles for ordered list.
Apply following classes to <ol> tag for alternate lists:
- list_ol-ua: Upper case alphabets
- list_ol-la: Lower case alphabets
- list_ol-ur: Upper case roman numerals
- list_ol-lr: Lower case roman numerals
Add attribute reversed to <ol> tag to reverse the order. Class list-ol can be used for styling the list.
