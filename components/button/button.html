Button

An important way of providing call to action. Buttons can have various styles and states.
Usually <a> tag or <button> tags are used to create buttons.
Use class btn.

Default

Any other kind of button can be created from this.
Following example includes the use of <button> tag and <a> tag for creating button.
Use class btn-default. Additionaly btn-sq and btn-cr can be used to obtain button having curved square corners and rounded ends respectively.
Use class link-noDecoration in addition to other required classes when using <a> to create button.

Default Button

Types

Different ways of styling a button. Often style of button is also used to show significance of the button.
Classes used:
btn-primmary: Primary button -- Most important button.
btn-outline: Outlined button -- Secondary importance.
btn-subtle: Subtle button -- Not extreamly visible.
btn-link: Link Button

Button With Icon

Has icon along with text. Add class btn-wtIcon to button with any other style.

Icon button

Icons act as buttons.
Use class btn-icon.

Floating action

It is removed from the flow of document and is suspended at a particular region of the webpage.
Use class btn-floating. Styling and position of this button can be changed using this class.
The 'back to top' button on botto-left corner of the webpage is an example of floating action button.

Block

Takes up 100% of width of parent container.
Use class btn-block along with other necessary classes.

Disabled

Used when no button click is intentended.
Use class btn-disabled along with other necessary classes.
