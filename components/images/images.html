Images

Many resposnive styles and style props can be applied to an image.

Responsive Images

Responsive images change dimensions according to the dimensions of parent container. This is important when the webpage is being viewed on different screen sizes.
One way to make images responsive is to avoid hard-coding the dimensions (height and/or width).
You can use following classes:
- img-100pr: Image takes 100% space available.
- img-75pr: Image takes 75% space available.
- img-50pr: Image takes 50% space available.
- img-25pr: Image takes 25% space available.
